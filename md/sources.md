# Sources

- SAP ERP MB51 flat-file export conventions (German column headers, DD.MM.YYYY dates)
- Utility portal CSV exports (billing period start/end, kWh usage, tariff codes)
- Concur/Navan-style travel CSV exports (segment types, IATA codes, cabin class)
- IATA airport code list (subset of major airports embedded locally for demo)
- SI unit conversion references (kWh, L, kg, km conversions)

## Sample Data Rationale
- SAP sample includes mixed units and a bad quantity to exercise flagging.
- Utility sample includes multi-month billing periods and large kWh values.
- Travel sample includes air, hotel, car, and rail segments.
